Borrowing heavily from [this project](https://github.com/tariquesani/beanconqueror-to-obsidian).  
  
Copy the config-example.yaml file to config.yaml and use that for config.
Ensure that the config is set up to point to the unzipped file from your twitter archive called 'tweet.js'. (This project was based on an archive extracted from Twitter in 2020 - its format may have changed since then.)  
Run main.py.  
It will outputs markdown files to the configured output directory.
